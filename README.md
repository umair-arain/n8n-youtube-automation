# n8n-youtube-automation
# 🤖 Automated YouTube Shorts & Reels Generator (AI Pipeline)

A fully automated backend automation pipeline that fetches content from Google Sheets, dynamically generates high-quality vertical videos (Shorts/Reels), and publishes them directly to YouTube using **n8n**, **Creatomate**, and **Google Cloud APIs**.

## 🚀 Key Features

*   **Zero-Human Intervention:** Automated content sourcing directly from database/spreadsheets.
*   **Dynamic Video Rendering:** Automated video compilation including custom text overlays, transitions, and backgrounds via rendering APIs.
*   **Asynchronous Processing:** Smart synchronization logic utilizing state-management (Wait Nodes) to handle rendering buffer times flawlessly.
*   **Automated Publishing:** Instant multi-platform distribution utilizing official YouTube Data API v3 with pre-configured metadata (Titles, Tags, Categories).

---

## 🛠️ Tech Stack Used

*   **Workflow Automation:** n8n (Advanced Node Orchestration)
*   **Content Database:** Google Sheets API
*   **Video Generation Engine:** Creatomate API / High-throughput Video Rendering
*   **Video Destination:** YouTube Data API v3 (OAuth2 Verified App System)
